# 🍎 Análise de Dados: Divino Pomar

Projeto de análise de dados focado na estruturação, tratamento e visualização de informações operacionais da **Divino Pomar**. O foco deste projeto foi transformar dados brutos e desorganizados em insights estratégicos através de um pipeline automatizado.

## 🚀 Sobre o Projeto
A Divino Pomar enfrentava desafios com dados fragmentados que impediam a análise de desempenho. Este projeto resolveu essa dor através da implementação de um pipeline de ETL (Extração, Transformação e Carga) no Microsoft Excel.

## 🛠️ Tecnologias e Ferramentas
- **Microsoft Excel:** Ferramenta central para modelagem e visualização.
- **Power Query (Linguagem M):** Automação do tratamento de dados (ETL), garantindo repetibilidade e escala.
- **Tabelas Dinâmicas & Dashboards:** Criação de visualizações interativas para apoio à decisão.
- **QA (Quality Assurance):** Protocolo de auditoria para garantir 100% de integridade entre a base bruta e a tratada.

## 📊 Principais Resultados
- **Automatização:** Redução do tempo de fechamento de relatórios e eliminação de erros manuais.
- **Insights de Negócio:** Identificação de que 56% do faturamento provém de Assinantes Premium, permitindo estratégias direcionadas de fidelização.
- **Confiabilidade:** Dashboard auditado com sucesso, garantindo precisão nos dados apresentados à gestão.
